1 Practice Artefact for 2-3 Tree

1.1 Features

  • User can insert,search ,delete a node in a 2-3 tree
  • User can visualize the movement of nodes clearly along with viewing the required steps
  • User can control the speed of the animation

2.4 Instruction Box

  • The instruction box above the artefact carries information about how to use the artefact.
  • It has details over how the controls presented on the screen work.
  • A brief description of the usage of artefacts is also provided here. 
        <body onload="init();" class="VisualizationMainPage">

        <div class="instruction-box">
                <button class="collapsible">Instructions</button>
                <div class="content">
                        <b>Instructions</b>:
                        <ul>
                          <li>Click on <b>Generate Root</b> to generate a 23 tree.</li>
                          <li>Depending on the <b>Next element to be inserted</b> select one of the operations.</li>
                          <li>Click <b>Insert</b> if the element to be inserted does not cause any split.</li>
                          <li>Click <b>Insert Left</b> if the element gets inserted in the left of root</li>
                          <li>Click <b>Insert Middle</b> if the element gets inserted in the middle of root</li>
                  </ul> 
                </div>
        </div>

  • This div has the main content which renderes the algorithm specific control of the artefacts.
  • These controls are displayed in the form of buttons and check boxes.
  • Table for buttons to control specific animation (insert/find/etc).
  • Filled in by javascript code specific to the animtion.
  • Also has drawing canvas where all animation is done.

2.6 General Animation Control, Next Element and Comment Box

  • Table for buttons to control specific animation (insert/find/etc)
  • Filled in by javascript code specific to the animtion such as play, pause, skip e.t.c
  • Next Element div displays the the question, about the next element to be inserted into the tree.
  • The comment box displays a message to the user at each step while interacting with the artefact. 
                                <div id = "generalAnimationControlSection" style="display:none;">
                                        <table id="GeneralAnimationControls">  </table>         
                                </div>
                        </div> 
                </div>

                <script>
                        var collapsibleEl = document.getElementsByClassName("collapsible")[0];
                        collapsibleEl.addEventListener("click", function() {
                                        this.classList.toggle("active");
                                        var content = this.nextElementSibling;
                                        if (content.style.maxHeight){
                                                        content.style.maxHeight = null;
                                        } else {
                                                        content.style.maxHeight = content.scrollHeight + "px";
                                        }
                        });
    </script>

        </body>
</html>
